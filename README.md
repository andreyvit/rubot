Rubot
=====

Returns the current time for any given timezone

Usage
-----

Message or add [@rubot_bot](http://telegram.org) on telegram

```
/tz list
```

```
/tz list 3
```

```
/tz search asia
/tz search america
```

```
/tz America/Detroit
```

```
/tz Asia/Novosibirsk
```

```
/andrey
/mi
/joshua
/mal
```
